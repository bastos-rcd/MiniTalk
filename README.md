# MiniTalk

Application de chat avec Websocket

## Lancement du serveur

1. Installer les dépendances

   ```bash
   cd ws-server
   npm install
   ```

2. Démarrer le serveur

   ```bash
   npm run start
   ```


## Lancement de React 

1. Ouvrir un nouveau terminal

2. Installer les dépendances

   ```bash
   cd React
   yarn install
   # ou
   npm install
   ```

4. Démarrer l'application React

   ```bash
   yarn dev
   # ou
   npm run dev
   ```
